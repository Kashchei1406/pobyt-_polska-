# Настройка: данные с сайта → таблица + чат бота в Telegram

Сейчас заявки с сайта попадают в **Google Таблицу**. Чтобы они дублировались в **общий чат (группу)** в Telegram, нужно один раз настроить бота и подставить данные в скрипт.

---

## Шаг 1. Создать бота и получить токен

1. В Telegram откройте [@BotFather](https://t.me/BotFather).
2. Отправьте команду: `/newbot`.
3. Введите имя бота (например: **Pobyt Polska заявки**).
4. Введите username бота (должен заканчиваться на `bot`, например: `pobyt_polska_leads_bot`).
5. BotFather пришлёт **токен** вида: `7123456789:AAHxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx`
6. Скопируйте этот токен — он понадобится в шаге 3.

---

## Шаг 2. Добавить бота в общий чат и узнать chat_id группы

Чтобы заявки приходили в **общий чат** (группу):

1. Создайте группу в Telegram или откройте существующую.
2. **Добавьте бота в группу**: «Добавить участников» → найдите бота по username (из шага 1) → добавьте. При необходимости сделайте бота администратором, чтобы он мог писать в чат.
3. Напишите в группе любое сообщение (например: «Тест»).
4. Откройте в браузере (подставьте **свой токен** вместо `ВАШ_ТОКЕН`):
   ```
   https://api.telegram.org/botВАШ_ТОКЕН/getUpdates
   ```
5. В JSON найдите блок с сообщением из группы: `"chat": { "id": -1234567890, "title": "Название группы", ... }`.  
   **chat_id группы — это число с минусом**, например **-1001234567890**. Скопируйте его целиком (вместе с минусом).
6. Этот **chat_id** подставьте в скрипт в шаге 3 — тогда все заявки будут уходить в этот чат.

Если в ответе `getUpdates` массив `result` пустой — напишите в группе ещё одно сообщение и обновите страницу с `getUpdates`.

---

## Шаг 3. Вставить токен и chat_id в скрипт

1. Откройте проект в [script.google.com](https://script.google.com) (тот, куда вставлен код из `backend/FormToSheetAndTelegram.gs`).
2. В начале файла найдите строки:
   ```javascript
   const BOT_TOKEN = 'ВАШ_ТОКЕН_БОТА';
   const CHAT_ID = 'ВАШ_CHAT_ID';
   ```
3. Замените на свои значения (в кавычках):
   ```javascript
   const BOT_TOKEN = '7123456789:AAHxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx';
   const CHAT_ID = '-1001234567890';
   ```
   Токен — в кавычках. **CHAT_ID** для группы — число с минусом, в кавычках (например `'-1001234567890'`). Для личного чата — положительное число.
4. Сохраните проект (Ctrl+S).

---

## Шаг 4. Опубликовать новую версию скрипта

1. В Google Apps Script нажмите **Развернуть** → **Управление развёртываниями**.
2. Нажмите карандаш (Изменить) у текущего развёртывания.
3. В поле **Версия** выберите **Новая версия**.
4. Нажмите **Развернуть**.

URL развёртывания не меняется — в `index.html` ничего менять не нужно.

---

## Готово

После этого при каждой отправке формы с сайта:

- строка по-прежнему добавляется в **Google Таблицу**;
- в **указанный чат** (группу или личный чат с ботом) в Telegram приходит сообщение с именем, телефоном, вопросом и источником заявки.

Проверка: отправьте тестовую заявку с сайта и убедитесь, что сообщение пришло в нужный чат.
