# Счётчик квиза и отчёт в Telegram в конце дня

## Что будет работать

- При завершении квиза сайт делает **+1** к счётчику (без сохранения ответов).
- В конце дня Google Apps Script отправляет в Telegram сообщение:
  - `📊 Квиз: прохождений за сегодня — N`

## 1) Обновить код в Google Apps Script

В проекте на `script.google.com` обновите код (вставьте актуальный `backend/FormToSheetAndTelegram.gs`), сохраните (Ctrl+S) и опубликуйте **новую версию** развёртывания.

## 2) Создать ежедневный триггер (таймер)

В редакторе Apps Script:

1. Слева откройте **Триггеры** (значок часов).
2. Нажмите **Добавить триггер**.
3. Выберите функцию: **`sendDailyQuizCount`**
4. Источник события: **По времени** (Time-driven)
5. Тип триггера: **Ежедневно** (Day timer)
6. Время: выберите **23:00–00:00** (например 23:00–24:00) — Apps Script запускает в окне, не в точную минуту.
7. Сохраните.

## 3) Проверка

- Пройдите квиз до конца пару раз.
- В Apps Script можно вручную запустить `sendDailyQuizCount` (кнопка ▶️) — сообщение должно прийти в Telegram с числом прохождений.

Примечание: счётчик хранится в `Script Properties` и **считается по всем пользователям**, которые проходят квиз на сайте.
